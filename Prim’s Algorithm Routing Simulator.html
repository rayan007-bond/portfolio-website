Prim’s Algorithm Routing Simulator

Github

The simulator provides an interactive way to understand how Prim’s Algorithm constructs an MST for network optimization. It uses GraphStream for dynamic graph visualization and JavaFX for UI.

The project simulates graph-based routing problems where nodes represent routers and edges represent communication links. Prim’s Algorithm is implemented to minimize total connection cost while ensuring full connectivity.

The challenge was integrating algorithm logic with GraphStream to provide real-time visual updates, while keeping the UI interactive and efficient with JavaFX.

The solution was to modularize the algorithm logic and link it with GraphStream’s graph visualization API. This ensured smooth updates when new edges were added to the MST, making the learning experience interactive.

Key Features

  • Graph Visualization with GraphStream
  • Prim’s Algorithm Implementation
  • Step-by-Step Simulation
  • JavaFX User Interface
  • Cost Calculation & Optimization
  • Educational Tool for Networking
