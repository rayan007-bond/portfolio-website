Tic-Tac-Toe Java Terminal-Based Application

Github

This is a text-based Tic-Tac-Toe game that runs in the terminal. Two players take turns entering their moves, with the program checking for wins, draws, and invalid inputs. It’s designed using clean Java code with a focus on OOP principles.

The game uses arrays to represent the board and implements modular methods for move validation, board display, and win condition checking. Players interact through console input, making it a simple yet effective demonstration of problem-solving in Java.

The main challenge was implementing efficient win-checking across rows, columns, and diagonals while keeping the program modular and easy to maintain.

The solution involved modularizing the logic with reusable functions for board updates, win/draw checks, and input handling. The program provides a smooth text-based gaming experience.

Key Features

  • Console-Based Gameplay
  • Two-Player Mode
  • Input Validation
  • Win & Draw Detection
  • Replay Option
  • Clean OOP Design
