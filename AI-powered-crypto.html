AI-Powered Crypto Signals Strategy App

Private Repository (Not Public)

This application acts as a crypto trading assistant, analyzing market trends in real-time using multiple indicators across different timeframes. The system follows a strict 5-layer confluence scalping strategy combining RSI, MACD, EMA, volume analysis, and candlestick patterns.

The app fetches live data from CoinGecko API, processes it with Python, and applies AI-driven strategies to generate actionable signals. It provides clear buy/sell recommendations along with confidence scores.

The challenge was ensuring accurate signals in a highly volatile crypto market, integrating multiple technical indicators without lag, and maintaining fast real-time responses.

The app was designed with a modular AI architecture where each layer of the strategy contributes to final decision-making. The signals are displayed in a clean Streamlit dashboard with visualization charts, making it trader-friendly.

Key Features

  • 5-Layer Confluence Strategy
  • Real-time data via CoinGecko API
  • AI-based signal confidence
  • Buy/Sell recommendations
  • Interactive Streamlit Dashboard
  • Advanced Chart Visualizations
