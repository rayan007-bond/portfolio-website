Certificate Sender – Streamlit App

Github

This application allows event organizers, universities, or institutions to send a single PDF certificate to all participants listed in a CSV file. Each recipient gets the same certificate file via email, along with a customized subject and message body.

The app reads recipient details (names & emails) from a CSV file and automates email delivery of a chosen PDF certificate. It streamlines distribution for large batches of participants, reducing manual work and errors.

Sending certificates manually via email is time-consuming, especially for events with hundreds of participants. The challenge was to design a user-friendly solution for non-technical users that automates the entire process.

Using Streamlit, the app provides a simple UI to upload a CSV file, select a certificate, and configure the email message. With Python’s smtplib, the certificates are sent securely to all recipients in just one click.

Key Features

  • Upload recipient list via CSV
  • Attach one PDF certificate
  • Customize subject & message
  • Bulk email sending
  • User-friendly Streamlit UI
  • Error handling & delivery status
